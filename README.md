# Farnen & Dermer Inc. — Demo Website

A demo marketing website for a fictional Baltimore HVAC company, built with an antique
Victorian-gentleman's-club aesthetic (deep forest green, mahogany, gold foil accents).

All company details, reviews, and history are fictional, created for demonstration purposes.

## Stack

- React 19 + Vite
- React Router (multi-page: Home, Services, Our History, Contact)
- Framer Motion (scroll reveals, parallax hero, animated stat counters, mobile nav)
- Imagery generated with Higgsfield AI, served from the Higgsfield CDN (`src/data/images.js`)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
